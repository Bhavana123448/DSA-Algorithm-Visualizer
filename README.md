# DSA Algorithm Visualizer

An interactive web application that helps visualize how sorting algorithms work step by step.

## Features
- Random Array Generator
- Bubble Sort Visualization
- Selection Sort Visualization
- Animated sorting bars

## Technologies Used
- HTML
- CSS
- JavaScript

## Algorithms Implemented
- Bubble Sort
- Selection Sort

## How to Run
1. Download or clone the repository
2. Open index.html in any web browser
3. Click "Generate Array"
4. Choose a sorting algorithm to visualize

## Project Purpose
This project helps beginners understand Data Structures and Algorithms by visually demonstrating how sorting algorithms work.
